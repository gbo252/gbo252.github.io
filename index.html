Netflix Random Movie Generator


About This Project

A web application that queries an API of up-to-date Netflix movies and displays a randomly selected movie. This project aims to make choosing a film from Netflix easier, by displaying one film at a time. This removes the issue of "too much choice" when scanning through the hundreds of available films on Netflix.
