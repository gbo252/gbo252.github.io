Flash Cards Online


About This Project

A full-stack React application that makes creating flash cards easy! Make a category and then start writing flash cards, with a front and back that can be hidden/revealed. The state is managed by Redux and the frontend is styled using Bootstrap. The backend is Node, Express, and MongoDB for the database. I am using Passport.js for authentication, and the browser's localStorage API for guest login.

Netflix Random Movie Generator


About This Project

A React web application that generates a random Netflix movie for your chosen Country and genre, removing the issue of "too much choice" when scanning through the hundreds of available films. The code base includes a full test suite using Jest. The app uses the unofficial Netflix API (uNoGS) for up-to-date movie data, and is styled using Bootstrap.

Built With

  • HTML
  • CSS
  • JavaScript
  • TypeScript
  • Bootstrap
  • React
  • Node
  • Jest
  • React Testing Library

Men @ 107


About This Project

An informational website for a men's gift shop called "Men @ 107" based in Muswell Hill, London. The site is fully responsive for varying screen sizes, uses the Instagram API for the photo slider, displays a contact form with a PHP backend and uses handlebars.js for reusable code on the "Products" page.
