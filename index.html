Flash Cards Online


About This Project

A full-stack web application that makes building flash cards in distinct categories easy! Sign in, create a category and start making flash cards consisting of a front and back which can be hidden/revealed. This was designed to be used for note-taking and revision purposes. The mongoDB back-end means you can login and retrieve your flash cards from anywhere!

Netflix Random Movie Generator


About This Project

A web application that queries an API of up-to-date Netflix movies and displays a randomly selected movie. This project aims to make choosing a film from Netflix easier, by displaying one film at a time. This removes the issue of "too much choice" when scanning through the hundreds of available films on Netflix.
